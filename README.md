# fxc-icon

Zero Dependencies Standalone SVG Icon selector encased in Native Javascript Custom HTMLElement

View at: <a href="https://fxc-icon.js.ie" target="_blank">fxc-icon.js.ie</a>

    <fxc-icon colour="#FF0000" icon="home" />
